API Documentation

To push a lead to Zoho via this proxy, use the following endpoint:

POST /api/leads/

Example Payload

{
    "Last_Name": "Doe",
    "First_Name": "John",
    "Email": "john.doe@example.com",
    "Company": "Example Inc",
    "Custom_Field_123": "Dynamic Value"
}

Note: If a field doesn't exist in Zoho, it will be automatically created as a text field.
